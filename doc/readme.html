Description:
You can use Chobi Dekha to build your web site's image gallery. It shows a small preview of the original image before launching the actual image. It will open the the larger version of the image in a popup window on demand. It loads only one thumbnail at a time. Thus, it saves time and your image gallery will load faster. All you have to make some simple changes. A graphical, JavaScript supported modern web browser is required to display this script. Chobi Dekha can run on all major graphical web browsers, like Microsoft Internet Explorer, Firefox, Netscape Navigator, Mozilla, Opera, Konqueror, Galeon, Safari, Camino and Epiphany.

Because this script is written entirely with JavaScript, it fully depends on the web browser, not on the operating system. So, you can view it on all modern graphical operating systems, like Microsoft Windows, Mac OS, Mac OS X, and several UNIX clones (Solaris, *BSD, Linux etc).

If you are curious to know, "Chobi" and "Dekha" are two separate Bengla words. "Chobi" stands for "Image/Art", and "Dekha" stands for "Viewing/Seeing". When we use them together, we actually mean enjoying a movie (more common).

Comments and feedbacks are very much appreciated regarding to this script. Thank you very much for using Chobi Dekha.




Installation:
Below you can find installation instructions for several operating systems. Please note that you will need a graphical, JavaScript (v1.3 and up) supported web browser to run this script.


On Windows- Unpack the .zip file with an archiving tool, like WinZip or WinRAR. Windows ME, XP and 2003 already got a built-in archiving tool. Now open the 'chobi_dekha.html' file with your favorite web browser.


On Linux- Unpack the .zip file with an archiving tool, like File Roller or unzip. Now open the 'chobi_dekha.html' file with your favorite web browser.


On other operating systems- Please follow the instructions provided for Linux.




Settings:
Open the 'chobi_dekha.html' file with a plain text editor (e.g. NotePad, gEdit, kEdit). Scroll down, you will see some well documented variables. You have to provide some information in the following sections for Chobi Dekha to work properly. 

/****************************************
 * Here you provide the location of the
 * first hi-resolution image, which will
 * be opened as soon as the user clicks
 * on the first thumbnail.
 ****************************************/
var url = "./img/river1.jpg";
Provide the location of the initial hi-resolution image inside the 'url' variable. This image will be opened in a popup window when the visitor clicks on the initial thumbnail. 
<!-- Provide the location of the thumbnail which should be displayed
     first. -->
<img src="./thumbs/river1.jpg" style="border: 1px solid #E9D000;"
     name="preview" border="0" width="150" height="100"
     alt="Click on the image to enlarge"></a>
Provide the initial thumbnail to display inside the <img> tag. It will be displayed as the first preview image of your image gallery as soon as the web page loads. 
<!-- Add more links (<a> tag) here and provide necessary arguments to
     loadImage() function. -->
<li><a href="#" class="c" onclick="loadImage('./thumbs/river1.jpg',
'./img/river1.jpg')">Rivers of Bangladesh (1)</a>
You should add 20 <a> tags if there are 20 images inside your image gallery. Provide the thumbnail and hi-resolution image's location inside the loadImage() function.
